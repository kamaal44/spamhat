To solve: Library not loaded: libmysqlclient.18.dylib (Mac OS X)

Symptom: After install mysql-server-x.x.dmg, when try to install DBD::mysql CPAN shows the message: ... Library not loaded: libmysqlclient.18.dylib

Try one of the next fixes

Fix 1:
cd /usr/lib/
sudo ln -sv /usr/local/mysql/lib/*.dylib .

Fix 2: 

add the next environment variable value to your .bash_profile or .bash_rc:

export DYLD_LIBRARY_PATH=/usr/local/mysql/lib:$DYLD_LIBRARY_PATH

Try to (re)install DBD::mysql and make sure all the tests are OK

NOTE: A local mysql server must be running in order to pass all tests
